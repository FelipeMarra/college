É nescessário instalar as dependências descritas no requirementes.txt. 
Recomendamos que isto seja feito dentro de um ambiente virtual.

Comando para ativar o ambiente virtual

.\venv\Scripts\activate

Comando para instalar os requerimentos

pip install -r requirements.txt

Para rodar o programa basta rodar a função main.py e a partir disso seguir os passos do menu apresentado.

A pasta datasets contem as intruções para baixar o dataset usado
